# Reading App

App móvil hecha con React Native y Spring Boot para leer libros en otros idiomas.

La idea es simple: subes un documento, la app lo convierte a texto y puedes ir guardando palabras o frases que no entiendas. Luego te muestra la traducción en tu idioma.

## Funcionalidades

- Subir documentos
- Convertirlos a texto
- Guardar palabras y frases
- Ver traducciones
- Lectura dentro de la app

## Tecnologías

- React Native
- Spring Boot
- API REST

## Cómo correr el proyecto

Clona el repo:
git clone <tu-repo>
cd <tu-proyecto>

Instala dependencias:
npm install

Inicia el servidor:
npm start

Corre la app:

Android:
npm run android

iOS:
npm run ios

## Notas

Asegúrate de tener el entorno de React Native configurado.  
En iOS toca instalar pods si es la primera vez.
